How to start backend-app

1. sudo docker run --rm --name work_db -e POSTGRES_PASSWORD=1 -e POSTGRES_USER=sergey1 -e POSTGRES_DB=work_db -d -p 5433:5432 -v $HOME/docker/volumes/postgres:/var/lib/postgresql/data postgres
2. sudo ss -lptn 'sport = :5432'
3. cd Postgre/
4. sudo kill 931
5. docker-compose up
